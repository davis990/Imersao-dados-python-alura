# Dashboard de Salários 

Foi criado um dashboard para explorar e fitrar dados de salários na área de dados. Este foi um projeto desenvolvido durante o curso de **"Imersão de Dados com Python da Alura"**

# O dashboard oferece:
- **4 gráficos interativos** para visualizar dados salariais
- **4 filtros dinâmicos** (Ano, Senioridade, Tipo de Contrato, Tamanho da Empresa)
- **Métricas resumidas** (Salário médio, máximo, total de respondentes, cargo mais frequente)
- **Comparação internacional** de salários para Data Scientists

# Filtros Interativos
Customize a análise selecionando:
- Ano(s) desejado(s)
- Nível(eis) de senioridade
- Tipo(s) de contrato
- Tamanho(s) de empresa

### Gráficos Disponíveis
1. **Top 10 Cargos** - Gráfico de barras com os maiores salários médios
2. **Distribuição de Salários** - Histograma mostrando faixas salariais
3. **Tipos de Trabalho** - Gráfico de pizza com proporção Remoto/Presencial/Híbrido
4. **Mapa de Salários** - Mapa interativo com salários de Data Scientists por país

## Aprendizados

Este projeto foi desenvolvido para praticar:
- Manipulação de dados com Pandas
- Criação de interfaces interativas com Streamlit
- Visualização de dados com Plotly
- Filtragem e agregação de dados
- Deploying de aplicações Python

## Tecnologias Utilizadas

- **Python 3.8+**
- **Streamlit** - Framework para criar aplicações web
- **Pandas** - Manipulação de dados
- **Plotly** - Gráficos interativos

# Ambientes de Desenvolvimento

- Notebooks: Google Colab (limpeza e filtragem de dados inicial)
